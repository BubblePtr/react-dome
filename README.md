# React Dome

一个开箱即用的 React + TypeScript + Tailwind CSS V4 脚手架项目。

## 特性

- ⚡️ Vite - 极速的开发体验
- ⚛️ React 18 - 最新的 React 版本
- 🎨 Tailwind CSS V4 - 最新的原子化 CSS 框架
- 📘 TypeScript - 类型安全
- 🔥 热模块替换 (HMR) - 即时反馈

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 项目结构

```
react-dome/
├── src/
│   ├── App.tsx          # 主应用组件
│   ├── main.tsx         # 应用入口
│   └── index.css        # 全局样式（Tailwind CSS）
├── index.html           # HTML 模板
├── package.json         # 项目配置
├── tsconfig.json        # TypeScript 配置
└── vite.config.ts       # Vite 配置
```

## 开始开发

直接编辑 `src/App.tsx` 文件即可开始开发你的应用。所有的 Tailwind CSS 类都已经配置好，可以直接使用。

## 技术栈

- [React](https://react.dev/) - UI 框架
- [TypeScript](https://www.typescriptlang.org/) - 类型系统
- [Vite](https://vitejs.dev/) - 构建工具
- [Tailwind CSS](https://tailwindcss.com/) - CSS 框架
