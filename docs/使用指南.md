# React Dome 使用指南

## 项目简介

这是一个开箱即用的 React + TypeScript + Tailwind CSS V4 脚手架项目。克隆后只需安装依赖即可开始开发。

## 快速开始

### 1. 克隆项目

```bash
git clone <your-repo-url>
cd react-dome
```

### 2. 安装依赖

```bash
npm install
```

### 3. 启动开发服务器

```bash
npm run dev
```

浏览器会自动打开 `http://localhost:5173`

### 4. 开始开发

直接编辑 `src/App.tsx` 文件，保存后浏览器会自动刷新。

## 可用命令

- `npm run dev` - 启动开发服务器
- `npm run build` - 构建生产版本
- `npm run preview` - 预览生产构建

## 项目结构

```
react-dome/
├── src/
│   ├── App.tsx          # 主应用组件（从这里开始修改）
│   ├── main.tsx         # 应用入口
│   ├── index.css        # 全局样式（Tailwind CSS）
│   └── vite-env.d.ts    # TypeScript 类型声明
├── index.html           # HTML 模板
├── package.json         # 项目配置
├── tsconfig.json        # TypeScript 主配置
├── tsconfig.app.json    # 应用 TypeScript 配置
├── tsconfig.node.json   # Node TypeScript 配置
└── vite.config.ts       # Vite 配置
```

## Tailwind CSS V4 使用

Tailwind CSS V4 已经配置完成，可以直接在 JSX 中使用所有 Tailwind 类名：

```tsx
<div className="flex items-center justify-center min-h-screen bg-gray-100">
  <h1 className="text-4xl font-bold text-blue-600">Hello World</h1>
</div>
```

## 常见问题

### 如何添加新的依赖？

```bash
npm install <package-name>
```

### 如何修改端口？

在 `vite.config.ts` 中添加：

```ts
export default defineConfig({
  plugins: [react()],
  server: {
    port: 3000
  }
})
```

### 如何配置路径别名？

在 `vite.config.ts` 和 `tsconfig.app.json` 中添加配置：

```ts
// vite.config.ts
import path from 'path'

export default defineConfig({
  plugins: [react()],
  resolve: {
    alias: {
      '@': path.resolve(__dirname, './src')
    }
  }
})
```

```json
// tsconfig.app.json
{
  "compilerOptions": {
    "paths": {
      "@/*": ["./src/*"]
    }
  }
}
```
